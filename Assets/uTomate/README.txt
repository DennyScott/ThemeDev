Welcome to uTomate!
=============================

Thank you for buying uTomate. You can find the quick start tutorial video and latest documentation
of uTomate on our website http://www.ancientlightstudios.com/utomate . 

You can find a "Demo Plan" automation plan in the "Demo Plan" folder. This plan simply prints
a "Hello World" message and the path to the current project to Unity's console. To run the plan, simply 
right-click it and select the "Run Automation Plan" option in the context menu. 

If you want to use uTomate in your team, please note that you need a one license of uTomate for each 
developer using it.

Should you require technical support you can shoot us a mail at support@ancientlightstudios.com .
